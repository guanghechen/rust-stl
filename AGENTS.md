# AGENTS.md

## 仓库定位

`rust-stl` 是 Rust `STL-style` 基础库：只做非业务的通用抽象。

核心目标：`高性能`、`高可靠`。

## 范围边界

包含：
- 通用数据结构、算法与组合抽象
- `iterator/collection` 基础能力
- 语义可定义的并发安全抽象

不包含：
- 业务/领域逻辑
- 框架绑定实现
- 应用层编排、服务流程与 UI 代码

## 开发原则

- 正确性优先于优化；性能主张必须有 benchmark 证据。
- 公共 API 必须写明复杂度、前置条件、后置条件与错误行为。
- 默认使用 safe Rust；新增 `unsafe` 必须附 `Safety` 说明与专项测试。
- 模块保持单一职责与单向依赖，禁止跨层耦合和循环依赖。

## 完成标准（DoD）

- 公共 API 文档完整（含复杂度）。
- 单测覆盖主路径、边界与错误路径。
- 性能敏感变更提供前后 benchmark 对比。
- 出现无法解释的性能回退，不合并。
